## About This Project
This interactive module demonstrates how to perform secure online payments, including:
- Two-Factor Authentication  
- One-Time Password (OTP)  
- QR Code Scanning  
- Viewing Balances  
- Creating Secure PIN  
- Online Safety for Elderly Users  

The website is hosted using **GitHub Pages**.
